# Test všeobecné odborné způsobilosti k získání Licence strojvedoucího

### Návod na instalaci
Soubory ke stažení jsou k dispozici pod releasem.

**Windows**
* Stáhněte soubor `voz.zip` a rozbalte do libovolného umístění.
* Spusťte `voz.exe`.

\
**Android**
* Stáhněte a nainstalujte soubor `voz.apk`.
* Spusťte aplikaci.

\
**Ostatní**
* Přejděte na Windows nebo Android a postupujte podle předchozích bodů.
